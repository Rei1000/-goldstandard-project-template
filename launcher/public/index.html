Goldstandard Launcher

Lokaler Wizard für den Goldstandard-Prozess: GPT-Phase (ChatGPT) und Agent-Phase (Cursor-Agent) mit strukturierter Übergabe.

Fortschritt / Phasen

  • GPT = Denken, Strukturieren, Handover erstellen
  • Agent = Umsetzung im Repository mit Handover-Kontext

Was jetzt zu tun ist

  1. GPT-Prompts 01 bis 05 in ChatGPT bearbeiten.
  2. Den erzeugten Projektkontext unten im Launcher speichern.
  3. Workflow-Input erzeugen und kopieren.
  4. Danach das Zielprojekt in Cursor öffnen und den Agent starten.

GPT-Prompts (ChatGPT)

Wähle einen GPT-Prompt aus, um den nächsten Schritt im ChatGPT-Flow zu sehen.

    Agent-Prompts (Cursor-Agent)

    prompts/agent/00-agent-task-frame.md ist eine verpflichtende Arbeitsregel und wird automatisch über den Workflow berücksichtigt. Du musst diesen Prompt nicht manuell starten.

    Als Start im Zielprojekt dient der zentrale Workflow prompts/agent/01-agent-run-workflow.md. Die Einzelprompts 01–04 kannst du bei Bedarf weiterhin hier lesen; für den kombinierten Einstieg musst du sie nicht mehr einzeln nacheinander starten.

    Einziger Startpunkt für die Ausführung ist prompts/agent/01-agent-run-workflow.md.
